はじめに

Ganpiは、Mac上で動作するプレーンテキスト向けの軽量テキストエディタです。 本ヘルプは、友人向けテストの参照用に最低限の情報をまとめたものです。

※ このヘルプはアプリに同梱され、メニューから既定のWebブラウザで開きます。

開く・保存

開く

File メニューから Open... を選択し、ファイルを開きます。 （ドラッグ＆ドロップ等を実装している場合はここに追記）

保存

File メニューから Save / Save As... を選択します。 文字コードや改行コードを変更した場合、保存時の挙動はここに明記してください。

検索・置換

Edit メニューから Find... / Replace... を選択します。 正規表現をサポートする場合、対応範囲（例：検索のみ／置換も可）をここに書きます。

例

検索:  foo([0-9]+)
置換:  bar$1

文字コード・改行コード

ステータスバーに現在の文字コード・改行コードが表示されます。 変更する場合は、該当メニューから選択します。

  • 文字コード：UTF-8 / UTF-16 / UTF-32（など）
  • 改行コード：LF / CRLF（など）

※ 変更可能な範囲は実装状況に合わせて調整してください。

設定

設定はアプリのポリシーに合わせて最小限にしています。 ここには、テストで確認してほしい設定項目だけを書いておくのが無難です。

  • フォント・フォントサイズ
  • ワードラップ
  • オートインデント
